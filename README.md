# 📊 Vibe Analysis: Análise Orientada por Curiosidade com IA

Bem-vindo ao repositório oficial do **Vibe Analysis**, o método prático que transforma perguntas em decisões orientadas a dados — sem a necessidade de saber programar ou dominar estatística avançada.

## 📌 O que é este material?
O Vibe Analysis é um **framework estruturado em 10 etapas** e organizado em cinco partes, desenhado para conduzir análises de dados complexas utilizando Inteligência Artificial (IA). Herdeiro direto da cultura do *Vibe Coding*, este guia metodológico oferece uma trilha clara de comandos (prompts) e lógicas para que qualquer profissional consiga investigar dados e extrair conhecimento acionável. 

O material inclui desde a formulação correta da pergunta de negócio, passando pelo diagnóstico estrutural da base de dados, até a criação de uma Narrativa de Decisão final.

## 🎯 Para que serve?
Este guia serve para dar autonomia analítica a gestores, líderes e profissionais que possuem acesso a dados urgentes, mas não têm formação técnica em **Ciência de Dados (Data Science)**. Ele permite:
* Utilizar IAs de mercado (como **ChatGPT, Claude e Gemini**) como analistas de dados altamente operacionais.
* Conduzir o raciocínio em três camadas analíticas essenciais: **Descritiva** (o que aconteceu), **Diagnóstica** (por que aconteceu) e **Preditiva** (o que significa para o futuro).
* Evitar alucinações da IA aplicando o ceticismo humano, mantendo o controle estratégico, a injeção de contexto e a **tomada de decisão** (Decision Making) sempre nas mãos do operador.
